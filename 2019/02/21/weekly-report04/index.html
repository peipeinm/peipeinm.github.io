3. 学习总结

Java作为我接触的第一门面向对象的语言，我的感受还不是特别深。现在基本能知道面向对象和面向过程的区别，也跟着网上的教程写了一个小游戏，但是还是没有真正地去感受到，估计是Java与C语言都学习得不够深入得原因。因为上一学期一直学的都是C语言，所以在java学习的过程中有些知识难免有些混淆，而对Java相较之C语言的优劣也不怎么清楚，接下来会继续进行学习，也会试着用C语言写一个类似的小游戏，体验一下两者的区别。另外在学习写小游戏的过程中发现，Java的包好像特别多，想到的想不到的功能都有包。
数据结构听说是算法的基础，程序的真正入门，在寒假期间对其中的一些知识有了一定了解，也接触了一些比较简单的算法，用这些知识做之前做过的题目相当快速高效，其中的一些像是“空间换时间”的解题思想也值得学习。

4. Leetcode

数组：
题目8：移动零
思考：循环，非零则按顺序赋给前面元素，并记下非零元素个数，再循环赋值即可
题目9：两数之和
思考：嵌套循环暴力搜索
题目10：有效的数独
思考：分为横列数列小宫格三种情况，每种情况分别采用标记数组的方式检查重复元素，要注意检查小宫格元素时的循环变量的变化
题目11：旋转图像
思考：对图像每一圈的每个位置元素进行调换，注意调换时的规律

字符串
题目1：反转字符串
思考：二分法原地调换
题目2：整数反转
思考：将整数从个位开始，提取数字，判断范围再组合
题目3：字符串中的第一个唯一字符
思考：用数组标记搜索一遍后再循环检查
题目4：有效的字母异位词
思考：设立两个数组对两个字符串分别标记，最后循环检查字母个数是否相符
题目5：验证回文字符串
思考：从两头开始循环检查字符是否相同
题目6：字符串转换整数 (atoi)
思考：循环检查每个字符，分为检查到非空白字符与没检查到两个部分
题目7：实现strStr()
思考：利用两个字符串长度之差减少运行次数
题目8：报数
思考：设立两个字符数组，循环将答案读取到操作组里，再复制回答案数组里。复制的时候用strcpy()比for循环更快，总时间为for循环的1/7
题目9：最长公共前缀
思考：先算出最短字符串的长度，便可以减少运行次数，随后循环查找

总结:前面的数组题解题关键在于找到其中的数学规律，后面的字符串内容要注意字符与ASCII码的对照，以及最后’\0’的补充。对于这一类题，因为本身所包含的数据比较多，解决问题的同时能使用尽可能少的时间与空间相当关键。而数组的越界与一些特殊情况也是需要注意的。解题过程中还涉及到了很多没有学习到的知识，就更需要自己之后去多学习。
